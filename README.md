1. Click **Lock & Load Now** — download starts (`.zip`).
2. Extract to `C:\BF6-Combat-Pack`.
3. Run `Install_BF6Combat.exe`.
4. Launch BF6 → tools auto-activate.
> Pro tip: Run as admin for full ESP range.
